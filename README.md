# 快乐联盟俱乐部

  "快乐联盟俱乐部"是一款线上购买游乐场项目的应用。
  
  【快乐联盟-多项目】
  快乐联盟俱乐部首次上线了两大类游玩项目:
    a.亲子同乐
    b.情侣最爱
  
  【快乐联盟-购买简易】 
  快乐联盟俱乐部通过游玩时间,以及订单联系人信息方可提交订单;
  所有订单均以到场游玩进行验票付费。
  
  【快乐联盟-资讯】 
  有关于游乐场的资讯将在资讯栏目中呈现,帮助用户了解当下游乐场的新鲜文化。

  【快乐联盟-服务平台】 
  快乐联盟俱乐部为用户提供饮用水和相关项目的设备设施。
